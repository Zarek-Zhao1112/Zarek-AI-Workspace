# Changelog

> 记录所有变更，方便追溯和复盘。

---

## 格式规范

```markdown
## [日期] 版本号

### 变更类型

- 新增：新功能或新内容
- 修改：更新已有内容
- 删除：移除内容
- 修复：修复问题

### 变更内容

- [模块] 具体变更内容
```

## 2026-06-29 v1.0.0

### 新增

- [README] 项目说明和使用方法
- [ABOUT_ME] 个人基本信息、优势、不足

- [INTERNSHIP] 实习工作内容和职责
- [PROJECT] AI运营分析平台项目详情
- [PROMPTS] 常用提示词模板
- [WRITING_RULE] 日报/周报写作规范

- [WEEKLY_GOAL] 周目标管理
- [CHANGELOG] 变更记录
