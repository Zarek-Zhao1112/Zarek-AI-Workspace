# Prompts

> 常用提示词模板，直接复制使用。

---

## 使用方法

1. 复制提示词模板
2. 填入你的具体信息
3. 粘贴到任何AI工具

---

## 日报提示词

```
- 负责80个Seller
- 主要工作：Seller运营、品牌招商、Seller分析、SKU治理、AI运营分析平台

1. [具体工作1]
2. [具体工作2]
3. [具体工作3]

- 复制版简洁专业，去掉标签词

```

## SellingPilot推销话术提示词

### 模板

```
请帮我生成SellingPilot推销话术。

## 店铺信息
- Seller ID: [填写Seller ID]
- 品牌: [填写品牌]
- SKU数: [填写SKU数]
- 履约方式: [填写SBS/SBN]
- 库存情况: [填写库存深度]
- 退货情况: [填写RMA%]

## 话术要求
- 商务语言，专业但不生硬
- 简洁口语化，像真人聊天
- 结合店铺实际数据
- 强调"99块一个月，三杯奶茶钱"
- 如果是多站点卖家，强调"统一管理所有站点"

## 输出格式
- 称呼
- 具体痛点
- 解决方案
- 行动引导
```

## 提示词使用技巧

### 1. 提供上下文

告诉AI你的背景、项目、目标，让它更好地理解你。

### 2. 明确需求

清楚地告诉AI你想要什么，包括格式、风格、内容等。

### 3. 给出示例

如果可能，给AI一个你喜欢的示例，让它模仿。

### 4. 迭代优化

第一次生成后，告诉AI哪里需要改进，逐步优化。

### 5. 记录好用的提示词

把你觉得好用的提示词记录下来，建立自己的提示词库。
