# Project

> AI运营分析平台项目详情。

---

## 项目定位

**名称：** AI运营分析平台

**定位：** AI辅助运营工具

**不是：**
- ❌ 聊天机器人
- ❌ Agent
- ❌ 创业项目
- ❌ 算法项目

**是：**
- ✅ 帮助运营分析Seller的工具
- ✅ 提升运营效率的辅助系统

---

## 核心功能

| 功能 | 说明 | 业务价值 |
|------|------|----------|
| Seller分析 | 分析卖家数据，识别异常 | 提前发现风险，避免平台罚款 |
| 库存分析 | 分析库存周转、滞销情况 | 优化库存结构，降低资金占用 |
| SKU治理 | 分析SKU数量、质量、销量 | 优化SKU结构，提升动销率 |
| Seller健康度 | 根据GMV、库存、SKU、品牌数量综合评分 | 量化卖家质量，指导运营决策 |
| AI招商助手 | 辅助品牌招商流程 | 提升招商效率，扩大品牌池 |
| 批量导入 | 批量处理卖家数据 | 减少重复工作，提升效率 |
| 数据可视化 | 图表展示数据 | 直观呈现问题，辅助决策 |

## 业务价值

### 对运营的价值

1. **效率提升**：自动化分析，减少手动工作
2. **风险预警**：提前发现异常Seller，避免罚款
3. **决策支持**：数据驱动的运营决策

### 对团队的价值

1. **标准化**：建立统一的分析流程
2. **可复用**：工具可以被其他运营使用
3. **知识沉淀**：将经验转化为系统
