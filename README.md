# Zarek AI Workspace

> 一个跨AI工具复用的工作说明书，帮助我高效使用AI助手完成工作。

---

## 仓库目的

让任何AI工具（Claude、Gemini、Cursor、Mimo、ChatGPT）都能快速了解我，帮我完成跨境电商运营工作。

---

## 使用方法

### 方式一：直接复制到AI工具

1. 打开需要的文件
2. 复制内容
3. 粘贴到AI工具的对话框
4. 告诉AI你要做什么

### 方式二：作为系统提示词

将 `ABOUT_ME.md` + `WRITING_RULE.md` 设置为系统提示词。

### 方式三：日报自动化

1. 告诉AI"今天做了XXX"
2. AI自动读取画像和写作规则
3. 生成分析版和复制版日报
4. 说"上传云端"，AI自动更新8个文件并推送到GitHub

**需要同步的8个文件**：
- CAREER.md（职业目标）
- ABOUT_ME.md（优势画像）
- CHANGELOG.md（变更记录）
- INTERNSHIP.md（实习工作记录）
- KNOWLEDGE.md（业务知识）
- PROJECT.md（项目详情）
- README.md（实习成果）
- WEEKLY_GOAL.md（周目标）

## 核心原则

1. **直接提醒，不迎合** - 发现方向偏差时直接指出
2. **多角度思考** - 站在导师/HR/老板三个角度分析
3. **体现业务价值** - 不是流水账，要说明为什么做、有什么用
4. **培养思维** - 帮助建立运营思维、数据思维、产品思维
