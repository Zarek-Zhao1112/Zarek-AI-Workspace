## Newegg Marketplace 基础知识

### 平台定位

- **类型：** 第三方卖家平台（Marketplace）
- **模式：** 卖家入驻，平台提供流量和基础设施
- **竞品：** Amazon、eBay、Walmart Marketplace

### 核心角色

| 角色 | 说明 |
|------|------|
| Seller（卖家） | 第三方商家，在平台上销售商品 |
| Operator（运营） | 平台方，负责Seller管理和运营 |
| BSD | 后台管理系统（BSD Support Portal） |

## Seller 运营知识

### Seller健康度指标

| 指标 | 说明 | 重要性 |
|------|------|--------|
| GMV | 销售额 | 核心指标 |
| RMA% | 退货率 | 风险指标 |

| 等级 | 标准 | 运营策略 |
|------|------|----------|
| A级 | 高GMV、低RMA | 重点维护 |
| B级 | 中等水平 | 培养提升 |
| C级 | 低GMV或高RMA | 重点关注 |

## SKU 治理知识

### SKU分析维度

| 维度 | 说明 | 分析方法 |
|------|------|----------|
| 数量 | SKU总数 | 统计分析 |
| 质量 | 动销率、滞销率 | 效率分析 |

### SKU优化策略

| 策略 | 适用情况 |
|------|----------|
| 保留 | 高动销、高利润 |
| 优化 | 中等水平、有提升空间 |
| 下架 | 低动销、高库存 |

## 经销商招募知识

### 4种找经销商方法

1. **展会名录** - CES、Computex等展会参展商
2. **亚马逊反查** - 通过亚马逊店铺反查经销商
3. **LinkedIn搜索** - 搜索行业关键词找经销商
4. **Google搜索** - 搜索品牌名+ distributor/wholesale

### 话术模板

**首次触达**：
> 您好，我是Newegg Marketplace的运营负责人。我们平台专注于3C数码硬件，月活跃用户超过5000万。了解到贵司在[品类]领域有丰富的渠道资源，想探讨合作可能性。

**跟进**：
> 您好，之前发送的合作资料不知您是否有看到？我们平台目前在[品类]品类有大量流量倾斜，合作模式灵活，期待与贵司进一步沟通。

### 关键业务认知

| 认知 | 说明 |
|------|------|
| AM三大核心职责 | PO控库存（库存管理）、观察调价格（竞品监控）、BD抓新品（业务开发） |
| 渠道商 vs 品牌方 | 渠道商决策链短、动力强，一个渠道商可带多个品牌SKU，是7月招商的潜在方向 |
| 定价分析方法 | 先抓竞品真实价格数据，再与卖家现价对比，用数据说话而非感觉 |
| 数据质量优先 | 分析结论的可信度取决于数据口径的一致性，修bug比加功能更重要 |
