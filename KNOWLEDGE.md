## Newegg Marketplace 基础知识

### 平台定位

- **类型：** 第三方卖家平台（Marketplace）
- **模式：** 卖家入驻，平台提供流量和基础设施
- **竞品：** Amazon、eBay、Walmart Marketplace

### 核心角色

| 角色 | 说明 |
|------|------|
| Seller（卖家） | 第三方商家，在平台上销售商品 |
| Operator（运营） | 平台方，负责Seller管理和运营 |
| BSD | 后台管理系统（BSD Support Portal） |

## Seller 运营知识

### Seller健康度指标

| 指标 | 说明 | 重要性 |
|------|------|--------|
| GMV | 销售额 | 核心指标 |
| RMA% | 退货率 | 风险指标 |
| SKU数量 | 商品丰富度 | 运营指标 |
| 库存周转率 | 库存效率 | 效率指标 |
| 品牌数量 | 品牌合作 | 扩展指标 |

### Seller分级

| 等级 | 标准 | 运营策略 |
|------|------|----------|
| A级 | 高GMV、低RMA | 重点维护 |
| B级 | 中等水平 | 培养提升 |
| C级 | 低GMV或高RMA | 重点关注 |
| D级 | 风险卖家 | 预警处理 |

## SKU 治理知识

### SKU分析维度

| 维度 | 说明 | 分析方法 |
|------|------|----------|
| 数量 | SKU总数 | 统计分析 |
| 质量 | 动销率、滞销率 | 效率分析 |
| 销量 | 销售分布 | 帕累托分析 |

### SKU优化策略

| 策略 | 适用情况 |
|------|----------|
| 保留 | 高动销、高利润 |
| 优化 | 中等水平、有提升空间 |
| 下架 | 低动销、高库存 |

## 数据分析知识

### 常用指标

| 指标 | 公式 | 说明 |
|------|------|------|
| 动销率 | 有销量SKU数/总SKU数 | 商品活跃度 |
| 库存周转率 | 销售成本/平均库存 | 库存效率 |
| 毛利率 | 毛利/销售额 | 盈利能力 |
| RMA% | 退货量/销售量 | 退货率 |

### 分析方法

| 方法 | 用途 |
|------|------|
| 趋势分析 | 观察指标变化趋势 |
| 对比分析 | 与基准或历史对比 |
| 异常检测 | 识别异常数据 |
| 帕累托分析 | 识别关键少数 |
